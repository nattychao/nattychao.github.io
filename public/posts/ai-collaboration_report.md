# AI 赋能 Web 开发：YEHWANG首页改版

## 1. 项目背景与目标
**演讲内容**：
大家好，今天我向大家汇报我们利用 AI 辅助编程技术完成的 Yehwang 个人主页重构项目。
我们的目标是将原本基于 Jekyll 的静态博客，升级为基于 Vue/Nuxt 的现代化 Web 应用，并实现多语言支持、动态壁纸库和极致的 UI 体验。
在这个过程中，AI 不仅仅是代码生成工具，更是我们的结对编程伙伴（Pair Programmer）。

---

## 2. 核心开发阶段回顾

### 第一阶段：架构重构与迁移 (Foundation)
**挑战**：将旧版 HTML/Jekyll 结构迁移到 Vue 组件化架构。
**AI 协作**：
我们首先确立了技术栈，并让 AI 协助我们将静态 HTML 拆解为可复用的 Vue 组件。

*   **提示词 (Prompt)**:
    > "重构 GitHub 主页为 Vue。我想创建一个美观且内容丰富的个人主页，作为技术博客和简历展示。"
    > "将静态 HTML 转换为 Vue。将全部内容...直接放入 Jewellery.vue，同时保持 1:1 的视觉和功能复刻。"

*   **成果**：
    *   成功搭建 Nuxt.js 项目框架。
    *   完成了 `Jewellery.vue` 等核心组件的 1:1 像素级复刻。
    *   解决了静态资源（Base64 图片）的迁移问题。

### 第二阶段：UI 交互与视觉升级 (Polish)
**挑战**：实现高端的视觉效果，如毛玻璃特效（Glassmorphism）和流畅的移动端交互。
**AI 协作**：
我们专注于细节打磨，通过 AI 快速试验复杂的 CSS 效果和动画逻辑。

*   **提示词 (Prompt)**:
    > "优化顶部导航栏毛玻璃效果。修改 CSS 以包含 backdrop-filter: blur(...) 和半透明背景色。"
    > "优化移动端菜单 UI。实现一个覆盖全屏高度的侧滑抽屉菜单...具有平滑的滑入/滑出动画。"
    > "为导航链接指示器添加动画。为底部选中指示器添加动画移动效果...平滑过渡其位置。"

*   **成果**：
    *   实现了随着滚动动态变化的毛玻璃顶部导航栏。
    *   开发了丝滑的 iOS 风格移动端侧滑菜单。
    *   添加了导航栏跟随鼠标移动的动态指示器，极大提升了用户体验。

### 第三阶段：功能开发与 API 集成 (Features)
**挑战**：接入外部 API 实现动态壁纸功能，并优化数据加载体验。
**AI 协作**：
AI 帮助我们处理了异步数据请求、跨域问题以及复杂的状态管理。

*   **提示词 (Prompt)**:
    > "壁纸 API 集成与分页。实现一个真实的 API 来获取壁纸数据...将 source 和 tagNames 数组映射到 API 参数。"
    > "持久化博客列表状态。确保在导航时保留博客列表状态（selectedCategory）...使用 Vue Router 管理状态。"

*   **成果**：
    *   成功对接 Infinity New Tab 壁纸 API，支持分类筛选和分页加载。
    *   解决了 `net::ERR_BLOCKED_BY_ORB` 等跨域请求难题。
    *   实现了博客列表的状态持久化，提升了浏览体验。

### 第四阶段：本地化与部署 (Localization & Deployment)
**挑战**：将全站内容中文化，并解决部署到 GitHub Pages 时的兼容性问题。
**AI 协作**：
利用 AI 的语言处理能力进行批量翻译，并利用其调试能力快速定位构建错误。

*   **提示词 (Prompt)**:
    > "将项目翻译成中文。将项目中的所有英文文本翻译成中文，以更好地适应中国地区。"
    > "修复 Vue 部署错误。修复部署的 GitHub 主页上出现的 'Failed to resolve module specifier vue' 错误。"

*   **成果**：
    *   完成了全站的中文化适配。
    *   修复了构建配置错误，确保网站成功上线运行。

## 3. 总结与展望
**演讲内容**：
通过本次项目，我们验证了 "人机协作" 模式在 Web 开发中的巨大潜力。
AI 帮助我们：

2.  **解难**：迅速定位跨域、构建等疑难杂症。
3.  **创新**：低成本尝试多种交互设计方案。

未来，我们将继续探索 AI 在自动化测试和性能优化方面的应用，打造更加完美的 Yehwang 网站。谢谢大家！
