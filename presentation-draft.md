# 演讲草稿模板

## 会议主题
[AI赋能Web开发：0代码创建网站]

## 演讲时长
[10分钟左右]

## 演讲目标
[人人都可以低成本，高效率开发网站]

我们先看成果，几个网站的UI效果：
1，yehwang首页改版（1~2天--->半天）
2，个人博客主站，AI 自主设计，自主开发，加以部分修改（3天--->半天）
3，my-jewellery网站（1小时创建完成）

以上都是AI+人工把关确认完成

接下来，我们以my-jewellery网站为例，看看AI是如何创建这个网站的。

第一步，选择合适的AI编程工具
这里推荐3款，用起来不错的。其中比较推荐Google的Antigravity，优化的比较好，生成的UI还原度高

第二步，项目的初始化，包括技术栈，环境配置等等，这里我们不关心，直接略过

第三步，重点，如何让AI帮我们生成网站页面（my-jewellery首页），整个开发过程我总结起来就一句话：把你想要的清楚的描述出来就行（无论是文字描述，还是直接喂图）然后微调+确认，最终把关的还得是自己
1，分析网站的结构，导航，bannner，分类，bannner，社交媒体，footer，这个分析过程，我直接丢给他一个首页截图，他自己分析完，然后创建了多个模块进行开发
2，每个模块给他设计图，他来根据图，生成对应的代码（分模块编码，是我实践出来还原度最高的方式）
3，现场我们简单演示一下这个AI编程的过程：（所有的模块都是这个工作流程）
现在的my-jewellery首页上面是一张固定的图片，现在我想让他变成轮播，无限循环，自动滚动，间隔时间为3秒：
提示词：
```
当前的banner是一张图片，现在变成轮播：

1，无限循环

2，间隔时间3秒

3，轮播的左右有两个翻页按钮，可以手动上一个，下一个轮播，鼠标移入轮播就会出现左右翻页按钮
```
其他的每个模块，都是这个操作步骤，只是描述的内容不同而已。

再比如：在header顶部增加一个黑色背景的的广告横幅，高度32px


最后自己把关确认，网站的UI效果符合预期，功能正常。即可上线使用。
